# 🔐 Microsoft Entra

## 📖 Overview

Microsoft Entra is Microsoft’s cloud-based **Identity and Access Management (IAM)** platform designed to manage identities, secure access, and protect organizational resources across cloud and hybrid environments.

Microsoft Entra was formerly known as **Azure Active Directory (Azure AD)** and is now part of a broader identity and security product family under Microsoft Entra.

---

# ⭐ Core Functions of Microsoft Entra

## 1️⃣ Authentication 🔑

Authentication verifies the identity of a user attempting to access a system or application.

### ✅ Common Authentication Methods
- 👤 Username and password
- 📱 Multi-Factor Authentication (MFA)
- 🔓 Passwordless authentication
- 🧬 Biometrics
- 🔐 FIDO2 security keys

---

## 2️⃣ Authorization 🛡️

Authorization determines what resources and services a user can access after successful authentication.

### 📌 Examples
- ☁️ Microsoft 365 access
- 🌐 Azure portal access
- 🏢 Internal company applications
- 🔗 Third-party SaaS applications

---

## 3️⃣ Identity Management 👥

Microsoft Entra manages:
- 👤 Users
- 👥 Groups
- 🎭 Roles
- 💻 Devices
- 📦 Applications

### ⚙️ Features
- ➕ User provisioning
- 👥 Group management
- 🛂 Role-Based Access Control (RBAC)
- 🔄 Self-service password reset

---

## 4️⃣ Single Sign-On (SSO) 🔗

Single Sign-On allows users to sign in once and access multiple applications without repeatedly entering credentials.

### 🎯 Benefits
- 😊 Improved user experience
- 🔑 Reduced password fatigue
- ⚡ Simplified access management

---

## 5️⃣ Multi-Factor Authentication (MFA) 📱

MFA adds an extra layer of security by requiring additional verification methods beyond passwords.

### 🔒 Examples
- SMS verification
- Authenticator app approval
- Fingerprint scan
- Security token

---

## 6️⃣ Conditional Access 🚦

Conditional Access applies security policies based on specific conditions.

### 📌 Conditions May Include
- 🌍 User location
- 💻 Device compliance
- 🌐 IP address
- ⚠️ Risk level
- 📱 Device type

### ✅ Example Policy
> Require MFA when users log in from outside the organization’s network.

Formerly Azure Active Directory (Azure AD)

### Functions
- Identity management
- Authentication
- Authorization

---

## 🌐 Microsoft Entra Internet Access
Secures internet traffic and web access for users and devices.

---

## 🔒 Microsoft Entra Private Access
Provides secure remote access to private applications without traditional VPNs.

---

## 🛡️ Microsoft Entra ID Governance
Helps organizations manage:
- User lifecycle
- Access reviews
- Privileged access
- Compliance requirements

---

## 🔍 Microsoft Entra Verified ID
Allows organizations to issue and verify digital credentials using decentralized identity technology.
